Portfolio de Simulación

Péndulo elástico

Un péndulo colgando de un muelle, simulado en este caso con integrador Runge-Kutta 4. Se desarrolló una clase Force que se puede ver al final del vídeo para facilitar el trabajo y permitir pintar el vector de cada fuerza en pantalla.

Fuegos artificiales

Simulación de fuegos artificiales usando sistemas de partículas. El viento se puede manejar con las teclas. Se ha usado un rellenado de fondo que no es totalmente opaco para que haya "rastro" y quede mejor visualmente. Además se utilizan otros pequeños sistemas de partículas para simular las chispas de los cohetes.

Colisiones partícula-partícula

Laboratorio realizado para el estudio del rendimiento de distintos sistemas de colisiones partícula-partícula. En este caso se muestra la diferencia de rendimiento entre un sistema de "todos contra todos" en comparación a un sistema grid y uno hash.

Superficie elástica

En este caso se trata de una superficie elástica formada por partículas unidas por muelles. Los muelles se pueden disponer de distintas formas para crear estructuras que se comportan de forma distinta. Además de estos muelles que sirven para simular el comportamiento de una tela o cama elástica, se generan muelles que impiden el contacto de la bola, con lo que la impulsan.
